# due-cloud

due-cloud是一个基于springboot2.7.0、springcloud、springcloudalibaba的一款企业级的快速搭建分布式项目的模板，里面包含的基础的认证授权等功能，是使用这可以快熟的搭建并使用，入门简单。使用springSecurity作为认证授权的基础，仿写springOauth2。提供多种授权方式：密码模式、手机验证码模式以及OpenId登录模式等。
